# News Portal - Simple Web App using Json Server

## How to Run

### Backend

`cd backend`

# This will run mongodb contrainer on local
``docker compose up -d``

# This will install all the dependencies
`npm i`

# This will run the server
`npm run start:dev`

### Frontend

`cd frontend`

# This will install all the dependencies
`npm i`

# This will run the server
`npm run dev`


## Endpoints

**Frontend**
`http://localhost:5173/`

**Server**
`http://localhost:3000/`
